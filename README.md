Descrição : Criar uma classe que receba uma sequência de bits para codificá-lo ou decodificá-lo e aplicar as operações matriciais.

Função: A diminuição de erros em codificações de bits é um desafio constante, e para isso é aplicado o método de codificadores em blocos.

Motivação: Aliar a programação em Python com a Engenharia de Comunicações, utilizando o software  GNURadio como interface.
